# 高考志愿数据仓库 Code Wiki

## 1. 项目概述

### 1.1 项目简介
本仓库是一个用于存储和管理中国两所著名师范大学高考志愿填报数据的项目。项目包含了华东师范大学和华中师范大学2023-2026年的招生计划、录取分数线、录取位次等详细信息。

### 1.2 项目用途
- 为考生提供志愿填报参考
- 为教育研究提供数据支持
- 便于学校招生信息的整理和分析

### 1.3 数据覆盖院校

| 4213 | 华中师范大学 | 湖北省 | 综合类 | 211、双一流、强基计划、师大六姐妹、101计划、公办 |

## 5. 数据使用指南

### 5.1 如何读取数据
可以使用各种编程语言或工具读取CSV数据：

#### Python示例
```python
import pandas as pd

# 读取华东师范大学数据
hdsfdx_df = pd.read_csv('hdsfdx.csv')

# 读取华中师范大学数据
hzsfdx_df = pd.read_csv('hzsfdx.csv')

# 查看前5行数据
print(hdsfdx_df.head())
print(hzsfdx_df.head())
```

#### R示例
```r
# 读取数据
hdsfdx_df <- read.csv('hdsfdx.csv')
hzsfdx_df <- read.csv('hzsfdx.csv')

# 查看数据结构
str(hdsfdx_df)
```

### 5.2 数据分析建议

#### 5.2.1 历年分数线趋势分析
可以分析同一专业历年的分数线变化，了解录取难度的变化趋势。

#### 5.2.2 专业组对比分析
对比不同专业组的分数线和计划数，为志愿填报提供参考。

#### 5.2.3 位次与分数的关系
结合录取分数和位次数据，更准确地评估录取概率。

### 5.3 注意事项
- 2026年的数据部分为0，表示数据尚未完全更新
- 不同年份的专业名称可能会有调整
- 特殊类型招生（公费师范生、优师专项等）有特定的报考条件和要求
- 选考科目要求是填报志愿的重要依据，请仔细核实

## 6. 项目维护

### 6.1 版本控制
本项目使用Git进行版本控制，所有数据变更都会有相应的提交记录。

### 6.2 数据更新流程
1. 收集最新的招生信息
2. 更新相应的CSV文件
3. 提交变更到Git仓库
4. 在CSV修改文件中记录修改内容（待定）

### 6.3 数据质量保证
- 确保数据来源的权威性
- 仔细核对录入的数据
- 定期检查和更新数据

## 7. 技术说明

### 7.1 文件格式
- 使用CSV（逗号分隔值）格式存储数据
- 编码格式为UTF-8
- 字段包含中文，需要正确处理编码

### 7.2 数据类型
- 数值型字段：计划数、分数、位次等
- 文本型字段：院校名称、专业名称、备注等
- 部分字段为空值，表示数据缺失或不适用
