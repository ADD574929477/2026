# CSV Helper 使用说明

## 快速开始

### 1. 创建新的学校数据

修改 `csv_helper.py` 中的 `create_college_csv()` 函数，复制并修改它来创建新的学校数据。

### 2. 使用 CSVHelper 类

```python
from csv_helper import CSVHelper

# 创建或加载文件
helper = CSVHelper('新学校名称.csv')

# 设置表头（与现有模板一致）
headers = [
    '院校ID', '院校名称', '地区', '类型', '院校标签', '行业标签', '所属部委',
    '专业组ID', '专业组名称', '专业组类别', '批次', '选考科目', '专业ID', '专业名称',
    '2026计划数', '2026最低分', '2026最低位次', '2026平均分', '2026最高分', '2026最高位次',
    '2025计划数', '2025最低分', '2025最低位次', '2025平均分', '2025最高分', '2025最高位次',
    '2024计划数', '2024最低分', '2024最低位次', '2024平均分', '2024最高分', '2024最高位次',
    '2023计划数', '2023最低分', '2023最低位次', '2023平均分', '2023最高分', '2023最高位次',
    '专业组标签', '专业组备注', '专业备注'
]
helper.set_headers(headers)

# 准备基础数据
base_data = {
    '院校ID': '学校ID',
    '院校名称': '学校名称',
    '地区': '所在地区',
    '类型': '学校类型',
    '院校标签': '标签1、标签2',
    '行业标签': '行业标签',
    '所属部委': '所属部门',
}

# 填充0到所有分数相关字段
for year in ['2026', '2025', '2024', '2023']:
    for field in ['计划数', '最低分', '最低位次', '平均分', '最高分', '最高位次']:
        base_data[f'{year}{field}'] = 0

# 添加专业组数据
row_data = base_data.copy()
row_data.update({
    '专业组ID': '专业组ID',
    '专业组名称': '专业组名称',
    '专业组类别': '物理组/历史组',
    '批次': '批次',
    '选考科目': '化学/思想政治等',
    '专业ID': '专业ID',
    '专业名称': '专业名称',
    '2025计划数': 10,  # 招生人数
    '专业组备注': '只招男生/只招女生',
    '专业备注': '专业说明文字',
})
helper.add_row(row_data)

# 保存和验证
helper.validate_fields()
helper.save()
```

## 主要功能

1. **按表头名称添加数据** - 不用担心字段位置，只需要按字段名提供数据
2. **自动填充0** - 自动把所有分数相关字段填0
3. **字段数量验证** - 保存前验证所有行的字段数量是否一致
4. **查看表头索引** - 使用 `helper.print_headers()` 查看所有表头和索引位置

## 建议

- 复制 `create_college_csv()` 函数来创建新的学校数据
- 先在一个小的测试文件上练习
- 使用 `validate_fields()` 确保数据正确
